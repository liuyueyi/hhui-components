trace-watch-dog
---

用于记录项目中，某一个链路的执行耗时情况，同时也支持将链路中的某些同步调用，改成异步执行，从而提高整体的性能表现

### 迭代记录

- [x] v1.0.1 支持在上下文中，随时获取 `TraceBridge` 进行手动埋点
- [x] v1.0.0 实现一个基础的 `TraceWatch` 工具类，通过手动埋点的方式记录整体的耗时分布情况
